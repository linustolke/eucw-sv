Instructions for volunteers

Important - This is not a request for translations. All texts must be written in the original language. It does not make sense to explain in English how to run a QSO e.g. in Russian. This means you can volunteer even if you hardly speak any English.
Format: Do not use particular templates for this collection, neither do we add the logo of EuCW to the QSO texts. Use plain html text.
Consider that all texts on eucw.org are in the public domain.
