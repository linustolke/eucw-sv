Instruktioner för frivilliga

Viktigt - Det är inte översättningar som önskas här. Alla dessa texter ska skrivas på originalspråk. Det är meningslöst att beskriva på svenska hur man genomför ett QSO till exempel på ryska. Det betyder att du kan anmäla dig som frivillig även om du inte förstår någon engelska.
Format: Använd inte någon speciell mall för denna sammanställning. Inte heller lägger vi in EUCWs logotyp på dessa QSO-texter. Använd vanlig enkel html-text.
Tänk på att all text på eucw.org tillhör public domain.
