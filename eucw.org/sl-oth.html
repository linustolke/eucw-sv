Instructions for volunteers

Just in case your language has no SL rules yet and you are willing to help, please get in touch with the webmaster (contact form).
We need HTML text for reading on the page and printable PDF.
Important - there is no need for a literal translation. It's even better to create a new text in your own words to help potential participants from your country.
Should you discover an error in one of the texts, please do the following:
1 - download the file containing the error
2 - correct the error in the file
3 - send back the corrected file
